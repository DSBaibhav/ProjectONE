# ProjectONE

## 📌 Overview
ProjectONE is a beginner-level project focused on learning and practicing the fundamentals of HTML. It contains multiple standalone HTML files demonstrating core concepts such as forms, tables, tags, and basic page structures.

---

## 🚀 Features
- Basic HTML page structure
- Form creation (single & multiple input forms)
- Table creation and formatting
- Tag usage and practice
- Simple layout demonstrations
